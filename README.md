# Book Tracker

**Автор:** Сидоров Дмитрий

## Описание

Book Tracker — десктопное приложение для учёта прочитанных книг с возможностью фильтрации и сохранения данных. Программа позволяет:


* добавлять книги с указанием названия, автора, жанра и количества страниц;
* просматривать список всех книг в табличном виде;
* фильтровать книги по жанру и количеству страниц;
* сохранять данные между сеансами работы;
* проверять корректность вводимых данных.

## Функциональность

**Основные возможности:**

1. **Добавление книг:**
   * поле «Название книги» — обязательное для заполнения;
   * поле «Автор» — обязательное для заполнения;
   * выпадающий список «Жанр» с предустановленными вариантами;
   * поле «Количество страниц» — должно быть положительным числом;
   * кнопка «Добавить книгу» — добавляет запись в таблицу.

2. **Фильтрация данных:**
   * по жанру (выбор из списка жанров);
   * по количеству страниц (>100, >200, >300, >500);
   * возможность комбинировать фильтры;
   * кнопка «Сбросить фильтр» — возвращает полный список.

3. **Управление данными:**
   * автоматическое сохранение при добавлении книги;
   * загрузка данных при запуске приложения;
   * отображение в таблице: название, автор, жанр, количество страниц.

4. **Валидация данных:**
   * проверка на пустые поля;
   * проверка, что количество страниц — положительное число.
